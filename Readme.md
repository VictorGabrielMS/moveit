## :octocat:Clone this repo

1. Clone :

```sh
  $ git clone https://github.com/VictorGabrielMS/moveit.git
```

2. Execute:

```sh
  $ cd moveit
```

3. Execute:

```sh
  $ npm install
  $ npm run dev
```
